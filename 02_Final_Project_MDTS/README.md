# 🚢 실전 프로젝트: MDTS (Ship Edge AI Integration)

### 📅 기간: 2026.03.20 ~ 2026.05.15

## 📝 프로젝트 소개
선박의 안전과 효율성을 극대화하기 위해 엣지 AI 기술을 접목한 통합 데이터 관리 솔루션입니다.

## 🚀 주요 특징
- **체계적인 설계**: 요구사항 정의서부터 테이블 명세서까지 완벽한 설계 산출물 보유
- **엣지 AI 통합**: 선박 데이터를 효율적으로 처리하기 위한 아키텍처 구상
- **데이터 중심**: RDBMS 및 NoSQL을 활용한 최적의 데이터 저장 구조 설계

## 🛠 기술 스택
- **Database**: SQL / NoSQL (MongoDB)
- **Design**: Powerpoint, Word, HWP
- **Concepts**: Edge AI, Software Engineering
