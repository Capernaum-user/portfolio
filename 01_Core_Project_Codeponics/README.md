# 🌿 핵심 프로젝트: 코드포닉스 (Code Powered Urban Harvest)

### 📅 기간: 2025.12.29 ~ 2026.02.13

## 📝 프로젝트 소개
아쿠아포닉스(Aquaponics) 시스템을 효율적으로 관리하기 위한 지능형 통합 관리 웹 솔루션입니다.

## 🚀 주요 기능
- **실시간 모니터링**: 센서 데이터를 실시간으로 시각화하여 수조 및 재배 환경 확인
- **지능형 대시보드**: 대시보드를 통한 직관적인 시스템 제어 및 상태 진단
- **일지 및 리포트**: 재배 과정의 데이터를 자동으로 수집하여 리포트 생성

## 🛠 기술 스택
- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Tools**: VS Code, Git

## 📂 폴더 구조
- /codeponics-web: 웹 프론트엔드 소스 코드
- /docs: 기획서, 설계서, 시연 영상 및 최종 발표 자료
